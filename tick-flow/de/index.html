Datenschutzrichtlinie

Letzte Aktualisierung: März 2026

Diese Datenschutzrichtlinie beschreibt, wie die mobile Anwendung TickFlow, entwickelt von Aners, mit Informationen umgeht.

Erhebung von Informationen

TickFlow sammelt, speichert oder überträgt keine personenbezogenen Daten.

Die Anwendung funktioniert vollständig offline und erfordert keine Registrierung oder Eingabe persönlicher Informationen.

Lokale Datenspeicherung

Die Anwendung kann kleine Datenmengen lokal auf Ihrem Gerät speichern, zum Beispiel Anwendungseinstellungen (z. B. ausgewählte Sprache oder Benutzereinstellungen).

Diese Informationen verbleiben ausschließlich auf Ihrem Gerät und werden nicht an externe Server übertragen.

Berechtigungen

Die Anwendung kann folgende Berechtigungen verwenden:

  • Vibration – wird verwendet, um Vibrationssignale für Timer-Ereignisse bereitzustellen.
  • Audio – wird verwendet, um Timer-Signaltöne abzuspielen.

Benachrichtigungen

TickFlow kann Benachrichtigungen anzeigen, um den Status laufender Timer anzuzeigen, während die Anwendung aktiv ist oder im Hintergrund läuft.

Diese Benachrichtigungen werden ausschließlich für die Timer-Funktion verwendet und sammeln oder übertragen keine personenbezogenen Daten.

Geräteinformationen

Die Anwendung kann grundlegende Geräteinformationen wie die Systemsprache lesen, um automatisch die passende Sprache der Benutzeroberfläche auszuwählen.

Diese Informationen werden ausschließlich auf dem Gerät verarbeitet und nicht an externe Server übertragen.

Drittanbieter-Dienste

TickFlow verwendet keine Analyse-, Werbe- oder Tracking-Dienste von Drittanbietern.

Datenschutz von Kindern

TickFlow sammelt wissentlich keine personenbezogenen Daten von Kindern oder Nutzern jeglichen Alters.

Änderungen dieser Richtlinie

Diese Datenschutzrichtlinie kann von Zeit zu Zeit aktualisiert werden. Alle Änderungen werden auf dieser Seite veröffentlicht.
